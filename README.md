🏥 Healthcare Analytics & Monitoring Portal

An interactive Healthcare Analytics Dashboard built using Python, Streamlit, Pandas, and Matplotlib to monitor hospital data, analyze diseases, and detect high-risk patients. The system provides secure admin login, patient management, and data visualization for better healthcare insights.

📌 Features
🔐 Secure Admin Login

Admin authentication using Email + Phone

OTP verification for secure access

OTP expiry timer for additional security

📊 Dashboard Overview

Displays total patients, departments, and diseases

Interactive bar chart and pie chart showing disease distribution

👤 Patient Profile

Search and view patient information using Patient ID

➕ Add Patient

Add new patient details including:

Age

Gender

Department

Disease

Heart Rate

Blood Pressure

Glucose

Admission Date

🏥 Department Analysis

View patient distribution across hospital departments

Explore department-wise patient data

🚨 Risk Alerts

Automatically detects high-risk patients based on:

High Blood Pressure

High Glucose levels

Abnormal Heart Rate

📅 Monthly Disease Report

Analyze disease cases by month

View patient data for specific diseases

🚻 Gender Based Disease Analysis

Analyze disease distribution across Male, Female, and Other

🛠️ Technologies Used

Python

Streamlit

Pandas

Matplotlib
